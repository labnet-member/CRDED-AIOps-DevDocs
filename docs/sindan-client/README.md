# SINDAN Client Documentation

このディレクトリは，**SINDAN クライアントセットアップ・出力データまとめ・デバッグ記録** に関するドキュメント群です．

---

## リポジトリ構成
```
sindan-clientt/
  ├ README.md                    ← このファイル
  ├ setup.md                     ← SINDAN クライアントのインストールと初期設定手順
  ├ (debug.md)                   ← 実行時の不具合調査・修正版の履歴（元コードがこの不具合に対応したため変更を取り消し済み）
  ├ other-changes.md             ← その他元のコードから変更した箇所
  ├ sindan-output-reference.md   ← 出力データの各フェーズ（Phase 0〜7）の定義と出力例
  │
  └ output-logs/                 ← 	各実行環境（Wi-Fi／Ethernet構成）ごとの実測ログ
```
